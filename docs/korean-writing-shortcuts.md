## 1. 한컴오피스 한글 (HWP)

한국에서 가장 많이 사용되는 워드프로세서로, 특히 한글 문서 작성에 최적화되어 있습니다.

### 파일 관련

| 단축키 | 기능 |
|--------|------|
| `Ctrl+N` | 새 문서 |
| `Ctrl+O` / `Alt+O` | 불러오기 |
| `Ctrl+S` / `Alt+S` | 저장하기 |
| `Alt+V` | 다른 이름으로 저장 |
| `Ctrl+P` | 인쇄 |
| `Ctrl+Q` / `Alt+X` | 끝내기 |

### 편집 관련

| 단축키 | 기능 |
|--------|------|
| `Ctrl+Z` | 되돌리기 (Undo) |
| `Ctrl+Shift+Z` | 다시 실행 (Redo) |
| `Ctrl+X` / `Shift+Del` | 오려두기 |
| `Ctrl+C` / `Ctrl+Ins` | 복사하기 |
| `Ctrl+V` / `Shift+Ins` | 붙이기 |
| `Ctrl+A` | 모두 선택 |
| `Ctrl+F` / `F2` | 찾기 |
| `Ctrl+H` / `Ctrl+F2` | 찾아 바꾸기 |

### 서식 관련

| 단축키 | 기능 |
|--------|------|
| `Ctrl+B` | 진하게 (Bold) |
| `Ctrl+I` | 기울임 (Italic) |
| `Ctrl+U` | 밑줄 (Underline) |
| `Alt+L` | 글자 모양 대화상자 |
| `Alt+T` | 문단 모양 대화상자 |
| `F6` | 스타일 |

### 글자 크기/간격 (한글 고유)

| 단축키 | 기능 |
|--------|------|
| `Alt+Shift+E` | 글자 크기 키우기 |
| `Alt+Shift+R` | 글자 크기 줄이기 |
| `Alt+Shift+W` | 자간 늘리기 |
| `Alt+Shift+N` | 자간 줄이기 |
| `Alt+Shift+K` | 장평 늘리기 |
| `Alt+Shift+J` | 장평 줄이기 |
| `Alt+Shift+A` | 줄 간격 늘리기 |
| `Alt+Shift+Z` | 줄 간격 줄이기 |

### 정렬

| 단축키 | 기능 |
|--------|------|
| `Ctrl+Shift+L` | 왼쪽 정렬 |
| `Ctrl+Shift+C` | 가운데 정렬 |
| `Ctrl+Shift+R` | 오른쪽 정렬 |
| `Ctrl+Shift+M` | 양쪽 정렬 |

### 입력/이동

| 단축키 | 기능 |
|--------|------|
| `Enter` | 문단 나누기 |
| `Shift+Enter` | 줄 나누기 (같은 문단) |
| `Ctrl+Enter` | 쪽 나누기 |
| `Home` | 줄 시작으로 |
| `End` | 줄 끝으로 |
| `Ctrl+Home` | 문서 처음으로 |
| `Ctrl+End` | 문서 끝으로 |
| `PgUp` / `PgDn` | 페이지 단위 이동 |

### 특수 기능

| 단축키 | 기능 |
|--------|------|
| `Ctrl+M,K` | 한자 변환 |
| `Ctrl+F10` | 특수 문자 |
| `F7` | 맞춤법 검사 |
| `F8` | 영문 자동 교정 |

**참고**: [한컴오피스 공식 단축키 일람](https://help.hancom.com/hoffice/multi/ko_kr/hwp/view/toolbar/shortcut(table).htm)

| `Ctrl+Shift+N` | 새 폴더 |
| `Ctrl+S` | 저장 |
| `Ctrl+Shift+S` | 모두 저장 |
| `Ctrl+,` | 환경 설정 |
| `Win+Alt+,` | 작업 설정 (프로젝트 설정) |

### 바인더 (문서 구조)

| `Ctrl+1` | 바인더 보기/숨기기 |

| 단축키 | 기능 |
|--------|------|
| `F11` | 컴포지션 모드 (집중 모드) |

| `Ctrl+K` | 스크리브닝 모드 (다중 문서 편집) |

### 서식

| 단축키 | 기능 |

### 탐색

| 단축키 | 기능 |
|--------|------|
| `Ctrl+G` | 이동 |
| `Ctrl+F` | 찾기 |
| `Ctrl+H` | 찾아 바꾸기 |
| `Ctrl+Shift+F` | 프로젝트 전체 검색 |

### 글자수/목표

## 4. 노벨라 (Novela)

한국 웹소설 작가를 위해 설계된 집필 도구입니다.

### 기본 편집

| 단축키 | 기능 |
|--------|------|
| `Ctrl+S` | 저장 |
| `Ctrl+Z` | 실행 취소 |
| `Ctrl+Shift+Z` / `Ctrl+Y` | 다시 실행 |
| `Ctrl+B` | 굵게 |
| `Ctrl+I` | 기울임 |
| `Ctrl+U` | 밑줄 |

### 노벨라 고유 기능

노벨라는 웹 기반 에디터로, 표준 단축키를 대부분 지원하며 다음 기능을 제공합니다:

- **스플릿뷰**: 설정 패널과 에디터를 나란히 배치
- **캐릭터 카드**: 인물 설정 관리
- **플롯 관리**: 스토리 구조 시각화
- **자동 저장**: 클라우드 자동 동기화

**참고**: [노벨라 공식 사이트](https://novela.so/)

## 6. 공통 단축키 패턴

대부분의 글쓰기 프로그램에서 공통적으로 사용되는 단축키입니다.

### 거의 모든 프로그램에서 동일

| 단축키 | 기능 |
|--------|------|
| `Ctrl+S` | 저장 |
| `Ctrl+Z` | 실행 취소 |
| `Ctrl+Y` 또는 `Ctrl+Shift+Z` | 다시 실행 |
| `Ctrl+X` | 잘라내기 |
| `Ctrl+C` | 복사 |
| `Ctrl+V` | 붙여넣기 |
| `Ctrl+A` | 모두 선택 |
| `Ctrl+F` | 찾기 |
| `Ctrl+H` | 바꾸기 |
| `Ctrl+B` | 굵게 |
| `Ctrl+I` | 기울임 |
| `Ctrl+U` | 밑줄 |
| `Ctrl+P` | 인쇄 |

### Mac 사용자 참고

Mac에서는 `Ctrl` 대신 `Cmd(⌘)`를 사용합니다:
- `Cmd+S` = 저장
- `Cmd+Z` = 실행 취소
- `Cmd+B` = 굵게

## 7. Storyforge 단축키 매핑

위 조사를 바탕으로 Storyforge에서 구현한/구현 예정인 단축키입니다.

### 현재 구현된 단축키

| 단축키 | 기능 | 비고 |
|--------|------|------|
| `Ctrl+S` | 수동 저장 | 자동 저장과 별개로 버전 생성 |
| `Ctrl+Z` | 실행 취소 | TipTap 에디터 내장 |
| `Ctrl+Y` | 다시 실행 | TipTap 에디터 내장 |
| `Ctrl+Shift+Z` | 다시 실행 | TipTap 에디터 내장 |
| `Ctrl+B` | 굵게 | TipTap 에디터 내장 |
| `Ctrl+I` | 기울임 | TipTap 에디터 내장 |
| `Ctrl+U` | 밑줄 | TipTap 에디터 내장 |
| `Ctrl+/` | 단축키 도움말 | Storyforge 고유 |
| `Ctrl+,` | 프로젝트 설정 | 스크리브너 참고 |
| `Ctrl+P` | 빠른 열기 | VS Code 스타일 |
| `Ctrl+H` | 버전 히스토리 | Storyforge 고유 |
| `Ctrl+Shift+N` | 새 프로젝트 | 일반적 패턴 |
| `Ctrl+Shift+F` | 프로젝트 전체 검색 | 스크리브너 참고 |
| `Ctrl+Shift+O` | 글자수 목표 | 스크리브너 참고 |
| `F11` | 집중 모드 | 스크리브너 컴포지션 모드 |
| `Escape` | 모달 닫기 | 일반적 패턴 |

### 향후 구현 고려 단축키

| 단축키 | 기능 | 참고 |
|--------|------|------|
| `Ctrl+F` | 찾기 | 현재 TipTap 내장 기능 사용 |
| `Ctrl+1/2/3` | 패널 전환 | 스크리브너 바인더 참고 |
| `Alt+Shift+E/R` | 글자 크기 조절 | 한글 참고 (고려 중) |
